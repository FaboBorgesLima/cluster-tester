# Prerequisites

All Kubernetes distributions documented where tested on the following baseline system requirements:

-   Debian 12.8.0
-   At least 2GB RAM for worker nodes
-   At least 4GB RAM for controller nodes
-   At least 2 CPU cores per worker node
-   At least 4 CPU cores per controller node
-   Root or sudo access
-   20GB disk space
-   Static IP addresses for all nodes
